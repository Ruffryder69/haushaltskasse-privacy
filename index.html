Kurzfassung

Die Haushaltskasse-App speichert alle deine Daten ausschließlich lokal auf deinem Gerät. Es werden keine personenbezogenen Daten erhoben, gespeichert oder an Dritte weitergegeben.

2. Welche Daten werden verarbeitet?

Die App verarbeitet ausschließlich Daten, die du selbst eingibst (z.B. Einnahmen, Ausgaben, Sparraten). Diese Daten werden lokal auf deinem Gerät im App-Speicher (AsyncStorage) abgelegt und verlassen dein Gerät nicht.

Es werden keine der folgenden Daten erhoben:

  • Name, E-Mail-Adresse oder andere Kontaktdaten
  • Standortdaten
  • Gerätekennungen (IMEI, Werbe-ID etc.)
  • Nutzungsstatistiken oder Analysedaten
  • Zahlungsdaten

3. Externe Datenabfragen (Live-Kurse)

Der Investitionsrechner kann auf Wunsch aktuelle Krypto-Kurse und Mining-Netzwerkdaten abrufen. Dabei werden folgende Drittanbieter-APIs kontaktiert:

  • CoinGecko API (coingecko.com) – aktuelle Kryptowährungspreise
  • Blockchair API (blockchair.com) – Netzwerk-Hashrate für BTC, ETH u.a.
  • Kaspa API (api.kaspa.org) – Netzwerk-Hashrate für KAS

Bei diesen Abrufen werden ausschließlich öffentliche Marktdaten abgefragt. Es werden dabei keine personenbezogenen Daten übermittelt. Es wird lediglich die IP-Adresse deines Gerätes durch den normalen HTTP-Abruf sichtbar – dies ist technisch unvermeidbar. Die Datenschutzerklärungen der jeweiligen Anbieter gelten ergänzend.

4. Datenspeicherung & Löschung

Alle eingegebenen Daten werden lokal im internen App-Speicher deines Gerätes gespeichert. Eine Übertragung in eine Cloud oder auf externe Server findet nicht statt. Die Daten können jederzeit durch Deinstallation der App vollständig gelöscht werden.

5. Weitergabe an Dritte

Es findet keine Weitergabe deiner Daten an Dritte statt.

6. Werbung & Tracking

Die App enthält keine Werbung und kein Tracking. Es werden keine Analyse-SDKs (z.B. Firebase Analytics, Crashlytics) verwendet.

7. Kinder

Die App richtet sich nicht gezielt an Kinder unter 16 Jahren. Es werden keinerlei Daten von Kindern erhoben.

Diese Datenschutzerklärung kann bei Bedarf aktualisiert werden. Die jeweils aktuelle Version ist stets unter dieser URL abrufbar.
